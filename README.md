# data-vis-charts

An interactive data visualisation project exploring how the popular sounds of UK rap and R&B evolved between 2010–2025 through streaming culture, algorithms, and globalisation.

Inspired by The Pudding’s *Music DNA* project:
https://pudding.cool/2025/04/music-dna/

---

# Project Overview

This project investigates one core question:

> How have the popular sounds of UK urban genres evolved from 2010 until now?

Using interactive web-based visualisations, the project explores:

- streaming-era music consumption
- globalisation of sound
- replayability and “loopable moments”
- changing song structures
- the influence of algorithms on chart music

The visualisations were created using:

- D3.js
- p5.js
- Readymag

---

# Dataset

The dataset was manually curated using:

- UK chart data
- Spotify audio characteristics
- Tunebat audio analysis

The dataset is divided into four eras:

- 2010–2013
- 2014–2018
- 2019–2022
- 2023–2025

Variables analysed include:

- BPM
- Energy
- Danceability
- Valence
- Duration
- Genre influence

---

# Features

## 🎧 Average Sonic Characteristics

Interactive p5.js visualisations comparing:

- BPM
- Energy
- Danceability
- Valence

Users can click to cycle through variables and compare how average sonic characteristics changed over time.

---

## 🌍 Globalisation of Sound

Interactive D3 stacked bar charts exploring:

- genre blending
- Afrobeats influence
- global streaming culture
- evolving UK urban sounds

Features include:

- hover tooltips
- animated transitions
- gradients
- responsive interaction

---

## 🔁 Loopable Moments & Song Structure

Waveform visualisations comparing:

- song duration
- hooks
- choruses
- structural pacing

Users can:

- play/pause songs
- compare older and newer song structures
- observe shifts toward replayability and shorter formats

- local CSV files
- audio files
- iframe embedding
